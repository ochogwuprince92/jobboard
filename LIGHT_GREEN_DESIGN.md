## 🎨 New Color Palette

### Primary Green Shades (More Visible)
```css
/* Light Green Tones */
#f0fdf4 - Very Light Green (Backgrounds)
#dcfce7 - Light Green (Gradients)
#d1fae5 - Pale Green (Borders)
#a7f3d0 - Soft Green (Hover states)

#059669 - Deep Green (AI badge)
```

### Where Green Shows Up

**1. Background** ✨
```css
/* Login/Register Page */
background: linear-gradient(135deg, #f0fdf4 0%, #dcfce7 100%);

/* Radial accents */
rgba(104, 211, 145, 0.15) - Light green glow
rgba(72, 187, 120, 0.12) - Soft green accent
```

**2. Cards** 💳
```css
/* Card border */
border: 1px solid rgba(104, 211, 145, 0.2);

/* Card shadow */
box-shadow: 
  0 8px 32px rgba(72, 187, 120, 0.12),
  0 2px 8px rgba(104, 211, 145, 0.08);

/* Hover */
border-color: rgba(104, 211, 145, 0.3);
box-shadow: 
  0 12px 48px rgba(72, 187, 120, 0.18),
  0 4px 12px rgba(104, 211, 145, 0.12);
```

**3. Input Fields** 📝
```css
/* Default */
background: #f0fdf4;
border: 1.5px solid #d1fae5;

/* Hover */
border-color: #a7f3d0;

/* Focus */
border-color: #6ee7b7;
box-shadow: 0 0 0 4px rgba(110, 231, 183, 0.2);
```

**4. Navigation Bar** 🧭
```css
/* Nav background */
background: linear-gradient(
  to bottom, 
  rgba(240, 253, 244, 0.98), 
  rgba(255, 255, 255, 0.95)
);
border-bottom: 1px solid rgba(110, 231, 183, 0.2);
box-shadow: 0 2px 8px rgba(72, 187, 120, 0.08);

/* Active link */
color: #10b981;
background: linear-gradient(135deg, #d1fae5 0%, #a7f3d0 100%);
box-shadow: 0 2px 8px rgba(16, 185, 129, 0.25);
```

**5. AI Tools Badge** 🤖
```css
/* Badge */
background: linear-gradient(
  135deg, 
  rgba(167, 243, 208, 0.3) 0%, 
  rgba(110, 231, 183, 0.25) 100%
);
border: 1px solid rgba(16, 185, 129, 0.3);
color: #059669;

/* Hover */
background: linear-gradient(
  135deg, 
  rgba(167, 243, 208, 0.5) 0%, 
  rgba(110, 231, 183, 0.4) 100%
);
border-color: rgba(16, 185, 129, 0.5);
color: #047857;
```

**6. Buttons** 🔘
```css
/* Primary button */

/* Hover */
```
